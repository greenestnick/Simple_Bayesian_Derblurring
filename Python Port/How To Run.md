# How to Run - Deblur Algo Python Port
1) make sure you have python 3 installed on the system
2) open up a terminal (or powershell) window in the 'Python Port' folder
3) type `python install_and_run_Deblur.py` in your terminal and hit enter
4) copy the FULL PATH of your test image 
5) when prompted, paste the FULL PATH of the image into the terminal and hit enter
6) it should generate output images in 'OUTPUT_IMG' grouped by run, may take a bit to run and will display feedback like RMS error rate in the terminal
